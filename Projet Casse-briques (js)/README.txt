PROJET PERSONNEL
    
Le but de ce projet était tout simplement de voir si je pouvais faire un projet majoritairement en JavaScript, que j'ai récemment appris, j'ai donc fait un Casse-briques.
Plus le joueur cassera toutes les briques, plus la balle ira vite (jusqu'à un maximum) et les briques vaudront de points.
Tous les 1500 points gagnés, le joueur recevra une vie supplémentaire.
    

PROJECT CODED BY ANTHONY VAUCHEL

PERSONAL PROJECT

The goal of this project was simply to see if I could create a project primarily in JavaScript, which I recently learned, so I made a Breakout game.
The more bricks the player breaks, the faster the ball will go (up to a maximum), and the bricks will be worth points.
Every 1500 points earned, the player will receive an extra life.
